    Firefox 153 enables Containers by default, and you can create your own - here's how
     
    Containers are a boon to your browsing security and privacy. Out of the box, Firefox now gives you four pre-configured containers: Personal, Work, Banking, and Shopping.
      Written by 
            Jack Wallen, Contributing WriterContributing Writer  July 22, 2026 at 10:11 a.m. PT                            Jack Wallen/ZDNETFollow ZDNET: Add us as a preferred source on Google.ZDNET's key featuresMozilla has moved Containers to an opt-out feature.Containers beef up both security and privacy.The developers also improved another Firefox security feature.The developers at Mozilla have decided to give their Firefox browser a bit of a security nudge with version 153. The security upgrade kicks off by enabling Containers by default. If you're unfamiliar with this important security feature, I wrote an introductory explainer on Firefox Containers. The gist is simple: Firefox containers isolate tabs from one another, so information from one tab cannot leak to another. This prevents sensitive data in one tab from being exposed on the off chance that another tab lands on a malicious site. Also, it prevents one site from snagging data from another tab to create a profile or fingerprint of you.Also: Firefox just fixed my biggest annoyance with web browsers - and others should copy ASAPAnd because Containers isolate cookies, you could even log into different accounts for the same service in one browser.Very handy.Firefox Containers are all about security and privacy, and they do a great job of fulfilling those imperatives.Prior to version 153, Containers were an opt-in feature, but now they're opt-out. Out of the box, you'll get four pre-configured containers: Personal, Work, Banking, and Shopping.
