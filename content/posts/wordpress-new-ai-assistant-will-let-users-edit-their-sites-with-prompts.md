<p class="has-text-align-none">Starting on Tuesday, WordPress users can edit their websites using the <a href="https://wordpress.com/blog/2026/02/17/wordpress-ai-assistant/">new AI assistant</a> built into the platform's site editor and media library, <a href="https://techcrunch.com/2026/02/17/wordpress-com-adds-an-ai-assistant-that-can-edit-adjust-styles-create-images-and-more/"><em>TechCrunch</em></a> reports. The AI has a sidebar in the WordPress site editor where users can ask it to edit and translate text, generate and edit images using Google's Nano Banana, and make adjustments to their sites like creating new pages or changing fonts.</p>
