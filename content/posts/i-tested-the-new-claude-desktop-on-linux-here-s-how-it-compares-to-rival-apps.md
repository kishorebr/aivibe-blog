    Claude Code finally has an official Linux desktop app. It's a great option, but trying to use local AI is where things get tricky.
      Written by

You might be surprised that, even in this age of AI, competitive open-source options are not hard to find. For example, I've been using both Alpaca and Moose for some time without issue. Both of these apps serve as GUIs for instances of locally installed Ollama (which is how I typically like to roll with AI). I've installed Ollama on most of my machines. Why? I'm not straining the power grid or compromising my privacy.
