Xcode 26.3 is now available as a release candidate. This newest version substantially increases Apple's coding intelligence features with built-in support for agentic coding. This will allow autonomous coding agents to work directly inside the IDE. A shift from assistants to agents Apple is positioning this update as a major expansion of the basic intelligence features first introduced in Xcode 26.
