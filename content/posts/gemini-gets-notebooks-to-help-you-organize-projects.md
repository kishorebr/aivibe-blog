<p class="has-text-align-none">Google's Gemini is getting a feature called "notebooks" to help you organize things about certain topics in a single place while using the AI chatbot, <a href="https://blog.google/innovation-and-ai/products/gemini-app/notebooks-gemini-notebooklm/">the company announced on Wednesday</a>. You can pull in things like files, past conversations, and custom instructions into notebooks that Gemini can then use as context while you're talking with it. </p>
<p class="has-text-align-none">Notebooks sound a lot like ChatGPT's Projects feature, which launched <a href="https://www.theverge.com/2024/12/13/24320800/openai-chatgpt-projects-folders-ai-chats">in 2024</a> and similarly <a href="https://help.openai.com/en/articles/10169521-projects-in-chatgpt">lets users</a> store things about a certain topic in one spot.
